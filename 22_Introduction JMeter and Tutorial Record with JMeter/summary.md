# Performance Test

### JMeter
- Perangkat lunak open source, aplikasi murni java
- Dirancang untuk memuat uji fungsional dan mengukur performa kinerja sebuah web
- Kelebihannya adalah open source, dan mudah untuk digunakan dengan GUI maupun konsol / non GUI

### JMeter Script
- TestPlan : Rencana besar test yang akan dilakukan
- Thred Groups : Kumpulan thread yang menjalankan skenario yang sama
- Samplers : Sebutan untuk request yang dikirim ke server
- Config elements : elemen yang digunakan untuk konfigurasi atau modifikasi sampler requests yang dikirim ke server
- Listeners : perekam data yang dihasilkan dari tes
- Timers : Fitur ini akan jalan terlebih dahulu sebelum fitur lain diuji
- Assertions : assert pada API/Web/Testing : merupakan kriteria tambahan apakah pass/tidak
- Pre-post processors : memproses response data sebelum/sesudah tes
